# Bar e Lanchonete Aquários - Site Institucional

Este é um site institucional estático de alta performance desenvolvido para a empresa **Bar e Lanchonete Aquários** (Joao Maria Alves dos Santos - ME - CNPJ: 81.379.901/0001-38). O projeto foi criado com foco em tradição, ambiente amigável e total responsividade.

## 🚀 Estrutura do Projeto

```text
aquarios-81379901000138/
├── index.html      # Página principal (Home, Sobre, Cardápio, Contato)
├── css/
│   └── style.css   # Estilização focada em robustez e tradição
├── js/
│   └── script.js   # Funcionalidades interativas (Scroll, Menu Mobile, Formulário)
└── README.md       # Documentação do projeto
```

## ✨ Funcionalidades

- **Design Premium:** Estética focada em tradição, ambiente de bar clássico e hospitalidade.
- **Totalmente Responsivo:** Otimizado para Mobile, Tablet e Desktop.
- **Scroll Suave:** Navegação fluida entre as seções da página.
- **Header Dinâmico:** Muda de aparência ao rolar a página para melhor legibilidade.
- **Menu Mobile:** Navegação intuitiva em dispositivos móveis.
- **Simulação de Formulário:** Validação básica e feedback visual de envio.

## 🛠️ Como Usar

1. **Edição de Conteúdo:** Abra o arquivo `index.html` em qualquer editor de texto (como VS Code) para alterar textos, imagens e informações de contato.
2. **Personalização de Cores:** As cores principais podem ser alteradas no arquivo `css/style.css` através das variáveis `:root`.
3. **Deploy:** Basta fazer o upload de todos os arquivos para qualquer servidor de hospedagem (HostGator, Hostinger, GitHub Pages, etc.).

## 📝 Notas de Desenvolvimento

- Desenvolvido em **HTML5**, **CSS3** e **JavaScript Puro**.
- Sem dependências de frameworks externos (Bootstrap, jQuery, etc.).
- Imagens otimizadas via Unsplash para carregamento rápido.
- Links externos configurados para redirecionamento padrão (`https://google.com`).
